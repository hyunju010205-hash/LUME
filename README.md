🌙 LUMÉ | Eternal Light  

*Will you still love me when I'm no longer young and beautiful...*  

---

프로젝트 개요  
LUMÉ는 "빛(Eternal Light)"을 모티프로 한 하이엔드 패션 & 뷰티 브랜드 웹사이트입니다.  
감성적인 럭셔리 브랜드 경험을 HTML, CSS, JavaScript를 기반으로  
정제된 구조와 미니멀한 디자인을 통해 구현했습니다.  

---

프로젝트 소개  
**LUMÉ | Eternal Light**은 "시간이 흘러도 변치 않는 아름다움"을 컨셉으로 제작되었습니다.  
정적인 이미지가 아닌, 감성적인 브랜드 무드와 부드러운 인터랙션을 통해  
명품 브랜드의 품격과 감정적 몰입감을 전달합니다.  

---

주요 특징  
- 풀스크린 히어로 섹션: 브랜드 시그니처 문구 중심의 메인 비주얼  
- 부드러운 스크롤 네비게이션: 자연스럽게 이어지는 페이지 전환  
- 반응형 레이아웃: 모바일, 태블릿, 데스크탑 완벽 대응  
- 컬렉션 갤러리: 시즌별 패션·뷰티 컬렉션 전시  
- 어바웃 & 컨택트: 브랜드 철학 및 뉴스레터 구독 기능  

---

기획 의도  
- 고급스러운 미니멀리즘을 기반으로 한 명품 감성 구현  
- “보는 순간 기억되는 브랜드”라는 콘셉트 실현  
- 디올(Dior), 셀린느(Celine) 등 하이패션 브랜드의 UX 흐름 분석 기반 제작  

---

기술 스택  
HTML5 / CSS3 / JavaScript (Vanilla)  
Google Fonts: Playfair Display, Inter  
이미지: Unsplash 고해상도 이미지 사용  
툴: VS Code, GitHub Pages  

---

설치 및 실행 방법  
1. 저장소 클론  
   ```bash
   git clone https://github.com/hyunju010205/lume-eternal-light.git
````

2. 프로젝트 폴더로 이동

   ```bash
   cd lume-eternal-light
   ```
3. index.html 파일을 브라우저에서 실행

폴더 구조

```bash
lume-eternal-light/

├── index.html
├── /assets
├── /css
│   └── style.css

    └── main.js
```
